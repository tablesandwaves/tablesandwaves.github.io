Experiments

These are a few simple melodic experiments, typically involving the mathematical and pattern manipulation on numeric representations of note sequences.

  • Automata Study: Phasing Voices Melodic lines that phase against each other to build a two voice melody
  • Automata Study: Guessing Game Interacting musical voices that build a two voice melody together
  • Automata Study: Chord / Melody Interaction Interacting musical voices that generate a system of chords and melodic accompaniment
  • RNBO + Typescript A simple FM duet built as a Web Audio synth & sequencer using RNBO
  • Monome Grid + Electron + Live Tutorial Algorithmic system for a Monome Grid and Ableton Live via Electron
  • Melodic Vectors Melody transforms, linear algebra style (sort of)
  • Infinity Series Centered melodic growth
  • Self Similarity Melodic self replication at N:1
